# Nikah Muyassar — Islamic Marriage Support Platform

> *"Marry the unmarried among you... If they should be poor, Allah will enrich them from His bounty."* — **Surah An-Nur 24:32**

A faith-driven digital platform connecting financially struggling Muslims who wish to marry with generous sponsors who fund their weddings.

**Live:** [https://nikah-muyassar.vercel.app](https://nikah-muyassar.vercel.app)

## Tech Stack

- **Frontend:** Next.js 14 App Router + TypeScript + Tailwind CSS
- **Backend:** Supabase (PostgreSQL + Auth + Realtime)
- **Storage:** Cloudinary
- **Email:** Resend
- **UI:** Custom shadcn/ui components + Framer Motion

## Getting Started

1. Clone the repo
2. Copy `.env.local` and fill in your credentials
3. Run `npm install && npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Deployment

Deploy to Vercel with one click — ensure all environment variables are set.
